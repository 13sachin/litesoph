============================
 LITESOPH
============================


Requirements
============

  * Python 3.7.6 or higher
  * Tkinter
  * click
  * Numpy
  * Matplotlib
  * Paramiko
  * scp
  * Rsync

Installation
============

.. code-block:: console

  $ git clone https://github.com/LITESOPH/litesoph.git
  $ pip install <path-to-litesoph>


Usage
=====

To start gui application, run:

.. code-block:: console

  $ litesoph gui
